0x02 Solved the problem of "Updates were rejected"
Because liaoxuefeng's blogs, we init local_db and configure github configuration, then we try use git_command to connect github like this:

1. git init //初始化仓库
2. git add "file.name" //添加文件到本地仓库
3. git commit -m "first commit" //添加文件描述信息
4. git remote add github + 远程仓库地址 //链接远程仓库，创建主分支
5. git push -u github master //把本地仓库的文件推送到远程仓库

之后，发现报错了。

正确步骤如下：
1. git init //初始化仓库
2. git add "file.name" //添加文件到本地仓库
3. git commit -m "first commit" //添加文件描述信息
4. git remote add github + 远程仓库地址 //链接远程仓库，创建主分支
5. git pull origin master --allow-unrelated-histories  // 把本地仓库的变化连接到远程仓库主分支
6. git push -u github master //把本地仓库的文件推送到远程仓库
